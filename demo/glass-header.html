Glass Morphism
Made Easy

Create stunning frosted glass effects using our Design System blur tokens. Scroll down to see the header effect!

Blur Level Comparison

SM - 8px

Subtle Blur

Light blur effect, great for subtle depth. Uses --blur-backdrop-sm

MD - 16px

Medium Blur

Balanced blur for most UI elements. Uses --blur-backdrop-md-2

LG - 24px

Strong Blur

Prominent glass effect. Uses --blur-backdrop-lg-3

XL - 40px

Maximum Blur

Intense frosted glass look. Uses --blur-backdrop-xl-4

/* Glass Header CSS */
.glass-header {
    /* Semi-transparent background */
    background: rgba(255, 255, 255, 0.1);
    
    /* Apply blur using token */
    backdrop-filter: var(--blur-backdrop-md-2);
    -webkit-backdrop-filter: var(--blur-backdrop-md-2);
    
    /* Subtle border for depth */
    border: 1px solid rgba(255, 255, 255, 0.2);
}
